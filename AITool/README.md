## About Portfolio
My portfolio is a **responsive and modern web design** that highlights my web development skills. It includes a hero section, a skills grid, and a contact section to provide a complete overview of my work and abilities.

---
## AI Tool Used
I used **Lovable AI** to generate the initial code and structure of the website.

---
## Features
- Fully **responsive design** (desktop, tablet, mobile)  
- Hero section with **headline, description, and contact button**  
- **Skills section** in a clean grid layout  
- Footer with **social media links**  
- Interactive **hover effects** on buttons  
- Elegant and minimal design
  
---
## Tech Stack
- **HTML**  
- **CSS3**  
- **JavaScript** (for mobile menu toggle)
   
---
## Changes Made
I customized the original AI-generated template to match my personal style:
- Changed **colors** (background, text, and buttons)  
- Updated **font family, font size, and font color**  
- Changed the **content** (headline, description, skills)  
- Adjusted **backgrounds** for sections  
- Modified **hover effects** of buttons for a smoother experience
  
---
## Conclusion
This project is for personal use and learning purposes. Feel free to use it as a reference.
